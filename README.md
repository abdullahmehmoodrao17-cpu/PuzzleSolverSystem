# 🧩 Puzzle Solver System - AI Mini Project

## 📌 Project Description
An intelligent autonomous agent that solves 8-puzzle and maze problems using BFS, DFS, and A* search algorithms with a Tkinter GUI.

## 🧠 Algorithms Implemented
- **BFS** (Breadth-First Search) - Guarantees shortest path
- **DFS** (Depth-First Search) - Memory efficient
- **A*** (A-Star Search) - Optimal with Manhattan heuristic

## 🖥️ How to Run
```bash
python main.py
